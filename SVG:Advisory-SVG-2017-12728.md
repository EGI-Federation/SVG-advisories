Title:       EGI SVG Advisory [TLP:WHITE] 'LOW' risk - XROOTD potential for remote code execution  [EGI-SVG-2017-12728]

Date:        2017-08-22
Updated:


Affected software and risk
==========================

LOW risk vulnerability concerning XROOTD

Package : XROOTD
CVE ID  : N/A
Bug ID  : N/A

A vulnerability was reported where unauthenticated remote code execution in some
configurations may be possible. However, there is no practical way to exploit this
vulnerability.  The faulty code is no longer being used.


Actions required/recommended
============================

Sites are recommended to update relevant components as soon as it is convenient.


Affected software details
=========================

This vulnerability has been fixed in xrootd version 4.6.1.


Component installation information
==================================

The official repository for the distribution of grid middleware for EGI sites is
repository.egi.eu which contains the EGI Unified Middleware Distribution (UMD).

http://repository.egi.eu/2017/08/10/release-umd-4-5-0/

Please note the EMI repositories are no longer maintained and may no longer be used.
